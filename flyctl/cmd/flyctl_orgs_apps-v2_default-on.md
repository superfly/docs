Configure this org to use apps v2 by default for new apps

## Usage
~~~
flyctl orgs apps-v2 default-on <org-slug> [flags]
~~~

## Options

~~~
  -h, --help   help for default-on
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --verbose               Verbose output
~~~
