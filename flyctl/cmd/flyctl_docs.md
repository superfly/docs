## About

View Fly documentation on the Fly.io website. This command will open a
browser to view the content.

## Usage

~~~
flyctl docs [flags]
~~~

## Options

~~~
  -h, --help   help for docs
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~
