Log of all issued SSH certs

## Usage
~~~
flyctl ssh log [flags]
~~~

## Options

~~~
  -h, --help         help for log
  -j, --json         JSON output
  -o, --org string   The target Fly organization
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --verbose               Verbose output
~~~
