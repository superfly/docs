List the certificates associated with a deployed application.

## Usage
~~~
flyctl certs list [flags]
~~~

## Options

~~~
  -a, --app string   Application name
  -h, --help         help for list
  -j, --json         JSON output
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --verbose               Verbose output
~~~
