Enable or disable WireGuard tunneling over WebSockets

## Usage
~~~
flyctl wireguard websockets [enable/disable] [flags]
~~~

## Options

~~~
  -h, --help   help for websockets
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~
