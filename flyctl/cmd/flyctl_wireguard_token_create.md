Create a new WireGuard token

## Usage
~~~
flyctl wireguard token create [org] [name] [flags]
~~~

## Options

~~~
  -h, --help   help for create
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --verbose               Verbose output
~~~
