Shows the list of regions the app is allowed to run in

## Usage
~~~
flyctl regions list [flags]
~~~

## Options

~~~
  -a, --app string   Application name
  -h, --help         help for list
  -j, --json         JSON output
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --verbose               Verbose output
~~~
