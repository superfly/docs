List snapshots associated with the specified volume.

## Usage

~~~
flyctl volumes snapshots list <volume-id> [flags]
~~~

## Options

~~~
  -h, --help   help for list
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~
