View a list of LiteFS Cloud regions

## Usage
~~~
main litefs-cloud regions [flags]
~~~

## Options

~~~
  -h, --help   help for regions
  -j, --json   JSON output
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output
~~~
