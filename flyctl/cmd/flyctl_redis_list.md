List Redis clusters for an organization

## Usage
~~~
flyctl redis list [flags]
~~~

## Options

~~~
  -h, --help         help for list
  -o, --org string   The organization to operate on
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~
