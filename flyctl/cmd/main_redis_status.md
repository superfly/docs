Show status of a Redis database


## Usage
~~~
main redis status <name> [flags]
~~~

## Options

~~~
  -h, --help   help for status
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output
~~~
