Lists organizations available to current user.


## Usage
~~~
flyctl orgs list [flags]
~~~

## Options

~~~
  -h, --help   help for list
  -j, --json   JSON output
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --verbose               Verbose output
~~~
