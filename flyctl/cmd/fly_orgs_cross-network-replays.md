Commands for managing cross-network replay permissions.

## Usage
~~~
fly orgs cross-network-replays [command] [flags]
~~~

## Available Commands
* [disable](/docs/flyctl/orgs-cross-network-replays-disable/)	 - Disable cross-network replays
* [enable](/docs/flyctl/orgs-cross-network-replays-enable/)	 - Enable cross-network replays
* [status](/docs/flyctl/orgs-cross-network-replays-status/)	 - Show cross-network replay status

## Options

~~~
  -h, --help   help for cross-network-replays
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output
~~~
