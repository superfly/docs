## About

View a list of regions where Fly has edges and/or datacenters

## Usage

~~~
flyctl platform regions [flags]
~~~

## Options

~~~
  -h, --help   help for regions
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~
