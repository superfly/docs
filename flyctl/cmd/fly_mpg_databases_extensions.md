Manage Postgres extensions in a managed postgres cluster database


## Usage
~~~
fly mpg databases extensions [command] [flags]
~~~

## Available Commands
* [disable](/docs/flyctl/mpg-databases-extensions-disable/)	 - Disable a Postgres extension on a database.
* [enable](/docs/flyctl/mpg-databases-extensions-enable/)	 - Enable a Postgres extension on a database.
* [list](/docs/flyctl/mpg-databases-extensions-list/)	 - List Postgres extensions for a database in an MPG cluster.

## Options

~~~
  -h, --help   help for extensions
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output
~~~
