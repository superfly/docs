## About

Displays the users email address/service identity currently authenticated and in use.

## Usage

~~~
flyctl auth whoami [flags]
~~~

## Options

~~~
  -h, --help   help for whoami
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~
