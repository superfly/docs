Show whether apps v2 is default on or off for the org

## Usage
~~~
flyctl orgs apps-v2 show <org-slug> [flags]
~~~

## Options

~~~
  -h, --help   help for show
  -j, --json   JSON output
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --verbose               Verbose output
~~~
