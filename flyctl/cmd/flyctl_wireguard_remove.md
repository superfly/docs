## About

Remove a WireGuard peer connection from an organization

## Usage

~~~
flyctl wireguard remove [org] [name] [flags]
~~~

## Options

~~~
  -h, --help   help for remove
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~
