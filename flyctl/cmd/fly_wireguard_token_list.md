List all WireGuard tokens

## Usage
~~~
fly wireguard token list [org] [flags]
~~~

## Options

~~~
  -h, --help   help for list
  -j, --json   JSON output
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output
~~~
