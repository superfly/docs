# _flyctl wireguard token start_

Start a new WireGuard peer connection associated with a token (set FLY_WIREGUARD_TOKEN)

### About

Start a new WireGuard peer connection associated with a token (set FLY_WIREGUARD_TOKEN)

### Usage
```
flyctl wireguard token start [name] [group] [region] [file] [flags]
```
