Show the currently authenticated user

## Usage
~~~
flyctl auth whoami [flags]
~~~

## Options

~~~
  -h, --help   help for whoami
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~
