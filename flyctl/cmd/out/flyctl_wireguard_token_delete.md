Delete a WireGuard token; token is name:<name> or token:<token>

## Usage
~~~
flyctl wireguard token delete [org] [token] [flags]
~~~

## Options

~~~
  -h, --help   help for delete
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~
