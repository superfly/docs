Show current Fly platform status in a browser


## Usage
~~~
flyctl platform status [flags]
~~~

## Options

~~~
  -h, --help   help for status
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~
