List DNS records within a domain

## Usage
~~~
flyctl dns-records list <domain> [flags]
~~~

## Options

~~~
  -h, --help   help for list
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~
