Add a WireGuard peer connection to an organization

## Usage
~~~
flyctl wireguard create [org] [region] [name] [flags]
~~~

## Options

~~~
  -h, --help   help for create
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~
