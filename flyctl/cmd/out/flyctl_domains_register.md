Register a new domain in an organization

## Usage
~~~
flyctl domains register [org] [name] [flags]
~~~

## Options

~~~
  -h, --help   help for register
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~
