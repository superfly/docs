Delete an existing organization.


## Usage
~~~
flyctl orgs delete [-yes] [slug] [flags]
~~~

## Options

~~~
  -h, --help   help for delete
  -y, --yes    Accept all confirmations
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output
~~~
