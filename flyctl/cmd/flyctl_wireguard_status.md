Get status for a WireGuard peer connection

## Usage
~~~
flyctl wireguard status [org] [name] [flags]
~~~

## Options

~~~
  -h, --help   help for status
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --verbose               Verbose output
~~~
