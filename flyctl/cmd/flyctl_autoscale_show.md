V1 APPS ONLY: Show current autoscaling configuration

## Usage
~~~
flyctl autoscale show [flags]
~~~

## Options

~~~
  -a, --app string   Application name
  -h, --help         help for show
  -j, --json         JSON output
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --verbose               Verbose output
~~~
