Runs the Synthetics agent in the foreground.

## Usage
~~~
main synthetics agent [flags]
~~~

## Options

~~~
  -h, --help   help for agent
~~~

## Global Options

~~~
  -t, --access-token string   Fly API Access Token
      --debug                 Print additional logs and traces
      --verbose               Verbose output
~~~
